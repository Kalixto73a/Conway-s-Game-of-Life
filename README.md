## How does it work?

For those who don't know how Conway's Game of Life works, I'm going to explain it.

![Screenshot](https://res.cloudinary.com/dzfqdntdw/image/upload/v1775745815/images_yeljws.png)

It's a game with 0 players.

The game only has 4 rules:
1. If a cell is alive and has fewer than 2 other living cells around it, that cell dies.
2. If a cell is alive and there are exactly 2 or 3 other living cells around it, that cell will continue to live.
3. If a cell is alive and has more than 3 other living cells around it, that cell dies.
4. If a cell is dead and has exactly 3 cells around it, this cell will become alive.
